# EngPro Frontend

Frontend web/mobile demo cho do an He thong quan ly khoa hoc online, xay dung bang Expo + Expo Router.

## Chuc nang hien tai

- Home / landing page theo phong cach EngPro
- Login cho student flow
- Register voi 2 vai tro: nguoi hoc va giang vien
- UI student-first, login de he thong tu kiem tra vai tro tai khoan

## Cau truc chinh

- `app/`: route cua Expo Router
- `components/landing/`: giao dien trang chu
- `components/auth/login/`: module dang nhap
- `components/auth/register/`: module dang ky
- `components/auth/shared/`: layout dung chung cho auth
- `components/branding/`: logo va branding


## Cong nghe

- Expo
- Expo Router
- React Native
- TypeScript
- Expo Linear Gradient
- Expo Vector Icons
